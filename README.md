# EcoSol Technologies Website

## Deploy in 3 steps

### Option 1 — Vercel (Recommended, Free)
1. Push this folder to GitHub
2. Go to vercel.com → New Project → Import repo
3. Click Deploy — done in 60 seconds

### Option 2 — Run locally
```bash
npm install
npm run dev
# Open http://localhost:3000
```

### Option 3 — Build for any host
```bash
npm install
npm run build
npm start
```

## Add your domain
After deploying to Vercel:
1. Go to Project Settings → Domains
2. Add ecosoltech.in
3. Point your domain DNS to Vercel

## Update email addresses
Search for "info@ecosoltech.in" in the codebase and replace with your real email.
